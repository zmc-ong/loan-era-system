## Loan Eligibility and Risk Assessment System

### 📝 Description
A comprehensive Loan Eligibility and Risk Assessment System. This desktop application helps banks automate the loan approval process by analyzing customer financial data, credit history, and risk factors using algorithmic scoring models.

### 🚀 Features
- 👥 **Customer Registration** - Store personal and financial information, track employment history and income, credit history records
- 📝 **Loan Application Processing** - User-friendly application form, real-time data validation, automatic integration with risk engine
- 📊 **Risk Scoring Engine** - Multi-factor risk assessment, custom weighted scoring model, real-time risk category determination
- 👑 **Admin Dashboard** - View all pending applications, approve or reject loans, override automated decision when need, detailed risk breakdown per application
- 🔐 **User Authentication** - Secure login system, role-based access (Admin vs. Loan Officer)
